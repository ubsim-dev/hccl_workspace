趋势

  • baseline 小包 16MiB/rank 明显受启动和同步开销影响，只有 47.87GB/s；64MiB/rank 后升到 186GB/s 左右。
  • baseline 128-512MiB/rank 基本在 189-196GB/s 附近波动，1024MiB/rank 下降到 171.70GB/s，说明全局长尾对最终口径有明显影响。
  • MeshClos 固定平面 16MiB/rank 已有 236.51GB/s，64MiB/rank 后稳定在约 256-266GB/s，说明固定平面分配减少了 baseline 中同向链路竞争造成的损失。
  • 按 64-1024MiB/rank 的大数据量区间平均，固定平面约 261.1GB/s，baseline 约 186.9GB/s，提升约 39.7%。

配置说明

  • rank 数：128；group size：8。
  • 扫描数据量：16、64、128、256、512、1024 MiB/rank。
  • baseline：generated_topology_pod128_hccl_baseline_threadserial_a2a128_*，RTP multipath 和 packet spray 均开启。
  • 固定平面：generated_topology_pod128_hccl_meshclos2d_v3_strict_planestep_a2a128_*，mode=strict，traffic_order=v3-logical，dependency_mode=v3-plane-step-barrier。
  • 1024MiB/rank profile 已单独放在 baseline profile 和 固定平面 profile。
